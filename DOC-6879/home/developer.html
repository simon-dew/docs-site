Advanced Operations

  • Sub-document operations for efficient JSON manipulation

  • Async and reactive APIs for higher concurrency

  • Distributed transactions for multi-document ACID mutations

  • Indexing to improve query and search performance

  • Full-text search to power up language-aware, relevant searches and more

  • Analytics for complex queries and data analysis

  • Eventing for asynchronous triggers on data changes

Production Scale

  • Clustering of Couchbase server nodes for share-nothing, scale-out performance and availability

  • Auto-sharding and automatic placement of shard replicas

  • Replication between remote datacenters for geographic distribution

  • Advanced security with authentication, authorization, role-based access control, encryption, and auditing

Management Tools

  • Web management UI and admin REST API

  • Dynamic cluster membership and automated data shard rebalancing

  • Bulk data loading

  • Backup / restore

  • Deployment guidelines with security and sizing considerations

 

 

 

Couchbase Mobile and Edge

Couchbase Lite

 

  • Couchbase Lite is an embedded NoSQL JSON database for mobile and edge apps, and is available for Java, Android, Swift, Objective-C, C#, and JavaScript

Sync Gateway

 

  • Sync Gateway is an internet-facing server that securely transfers data between mobile and edge Couchbase Lite instances and Couchbase Server
