    Overview

    API reference guide for the Couchbase Capella Public API.

    You can download the specification for this reference as an OpenAPI file to import into tools such as Postman, Insomnia or similar: openapi.yaml

    Version : 2.0.0

    Host information

    https://cloudapi.cloud.couchbase.com
