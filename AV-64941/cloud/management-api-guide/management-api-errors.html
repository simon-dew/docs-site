Management API v4.0 Errors

  • reference
An explanation of error messages generated by the Capella Management API v4.0.

Error Messages

HTTP Status 400

Bad request.

Code Name Description HTTP Status

1000

BadRequest

The server cannot or will not process the request due to something that is perceived to be a client error.

Check if all the required params are present in the request body.

400

1000

BadRequestWithMessage

The request was malformed or invalid.

400

6004

InvalidBucketConflictResolution

The provided conflict resolution is not supported. Please choose a valid conflict resolution strategy for the bucket.

400

5001

JobActionEmpty

Invalid job configuration. Action field cannot be empty, and must be one of 'backup' or 'restore'.

Returned when a job is attempting to be added to a backup schedule with no action.

400

5002

JobSourceEmpty

Invalid job configuration. Source field cannot be empty, and must be one of 'manual' or 'scheduled'.

Returned when a job is attempting to be added to a backup schedule with no source.

400

5003

NilRestoreJobOptions

Invalid job configuration. All restore jobs must have restore options set.

Returned when a restore job is being added without restore options set.

400

5008

InvalidTimingDay

The day attribute is invalid. Only an integer between 0 and 6 is allowed.

Returned when the day attribute to the timing struct is not an integer between 0 and 6.

400

5009

InvalidTimingHour

The hour attribute is invalid. Only an integer between 0 and 23 is allowed.

Returned when the hour attribute to the timing struct is not an integer between 0 and 23.

400

5011

InvalidIntervalValue

The incrementalEvery attribute is invalid.

Returned when asked to create or modify a backup schedule, but the incremental value provided is not valid. Must be one of '1', '2', '4', '6', '8', '12', or '24'.

400

5012

InvalidIntervalUnit

The unit attribute is invalid. Must be one of 'minute', 'hourly', 'daily', or 'weekly'.

Returned when the unit attribute is not one of the expected allowed constants.

400

5013

InvalidFullIntervalUnit

Unsupported type for backup schedule. The full backup type must be 'weekly'.

Returned when the scheduled backup type is not 'weekly'.

400

5014

InvalidIncrementalIntervalMinimum

Invalid incremental backup interval. The minimum allowed incremental backup interval is every 4 hours.

Returned when the incremental backup interval is quicker than every 4 hours.

400

5015

InvalidBackupRetentionTime

The backup retention time is invalid.

Returned when the given backup retention time is not supported.

400

HTTP Status 401

Unauthorized.

Code Name Description HTTP Status

1001

Unauthorized

The request is unauthorized. Please ensure you have provided appropriate credentials in the request header. Please make sure the client IP that is trying to access the resource using the API key is in the API key allowlist.

401

HTTP Status 403

Forbidden.

Code Name Description HTTP Status

1002

AccessDenied

Access Denied.

Your access to the requested resource is denied. Please make sure you have the necessary permissions to access the resource.

403

1003

RBACAccessDenied

Access Denied.

Make sure you have adequate access to the resource.

403

2001

ProjectCountQuotaReached

Unable to create a project. Project count limit is reached.

Please ensure to stay within the quota. If required, please create support ticket in order to increase the quota.

403

HTTP Status 404

Not found.

Code Name Description HTTP Status

2000

ProjectNotFound

The server cannot find a project by its ID.

Check if the project ID is valid.

404

2002

ProjectsNotFoundWithMessage

Check if the project ID is valid.

404

3000

NotFound

Please ensure that the organization ID is correct.

404

4025

ClusterNotFound

The requested cluster details could not be found or fetched. Please ensure that the correct cluster ID is provided.

404

4026

V3ClusterNetworkInfoNotFound

The cluster network information for the received ID could not be found. Please ensure that the correct ID is provided.

404

6008

BucketNotFound

The requested bucket does not exist. Please ensure that the correct bucket ID is provided.

404

5017

BackupRecordNotFound

The requested backup record could not be found.

Returned when the requested backup record could not be found.

404

5019

ExportRecordNotFound

The requested download record could not be found.

Returned when the download record cannot be found for the given ID and cluster.

404

5023

UnableToRetrieveBackup

Unable to retrieve the backup specified.

Returned when encountering an error while attempting to retrieve a backup record.

404

5029

BackupScheduleNotFound

Backup schedule not found.

Returned when a backup schedule is not found for the bucket.

404

8000

UserNotFound

Check userId is valid.

404

8013

DeleteUserNotFound

Verify the provided user ID and ensure it corresponds to an existing user before attempting to delete.

404

8018

UserWithIDNotFound

User not found. The system was unable to locate a user with the specified ID.

Ensure that the provided user ID is correct and corresponds to an existing user.

404

HTTP Status 409

Conflict.

Code Name Description HTTP Status

5000

JobAlreadyExists

Unable to add job, as it already exists and is waiting to be processed.

Returned when there is already a backup or restore job with the same key in the backup config schedule.

409

5018

BackupScheduleAlreadyExists

Unable to add backup schedule, as a schedule already exists for the bucket.

Returned when there is already a backup schedule for the bucket in the backup config schedule.

409

5031

BackupIDConflict

Unable to process request, as backup ID in the payload does not match backup ID in the path.

Returned when asked to restore a backup and backup IDs in path and payload do not match.

409

412
