Couchbase Docs

Couchbase Server is an open source, JSON document database that brings together NoSQL and SQL++. It provides a scale-out, shared-nothing, auto-sharded, replicated document store with a managed cache, SQL++ querying, indexing and other capabilities.

For mobile and edge environments, Couchbase Lite is a native, on-device, embeddable database that can synchronize data through Couchbase Sync Gateway to one or more clusters of Couchbase Server.
